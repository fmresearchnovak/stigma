Directed Graph Example (Mermaid.js)

This page demonstrates how to display a directed graph using the Mermaid.js library. You define the graph using simple text syntax, and Mermaid renders it into an SVG image.

This flowchart shows a basic decision process.

More Complex Graph with Subgraphs

graph TD
    subgraph Data Flow
        DB((Database)) --> API[REST API];
        API --> APP(Web Application);
    end

    subgraph User Interaction
        USER[User] -- interacts with --> APP;
        APP -- sends data to --> MQ[Message Queue];
    end

    MQ --> Processor(Data Processor);
    Processor --> DB;

    click APP "https://www.example.com" "Visit Example App"

This graph illustrates a more complex system with data flow and user interaction subgraphs.

Custom Styling and Icons

graph TD
    A[Christmas] -->|Get money| B(Go shopping)
    B --> C{Let me think}
    C -- One --> D[Laptop]
    C -- Two --> E[iPhone]
    C -- Three --> F[fa:fa-car Car]
    D --> G((End))
    E --> G
    F --> G

    style A fill:#f9f,stroke:#333,stroke-width:2px

    style C fill:#ccf,stroke:#000,stroke-width:2px,stroke-dasharray: 5 5
